## 三、种子用户群运营

### 3.1 微信群建立

**群名**：「AI跨境视频 内测群」
**入群条件**
- 正在经营 TikTok Shop 或 Amazon
- 有视频制作需求
- 愿意每周提供20分钟反馈

**群规则**
1. 内测期间完全免费，但需要主动反馈
2. 发现bug请截图发群内，有奖励
3. 每周五有30分钟语音反馈会
4. 严禁广告，违者移除

**群运营节奏**
| 时间 | 内容 |
|------|------|
| 每天早上 | 发当日更新日志（小版本迭代） |
| 每天下午 | 分享1个跨境干货 |
| 每周三 | 发起投票：下周优先做哪个功能 |
| 每周五 | 语音反馈会，整理需求 |

### 3.2 种子用户激励体系

| 行为 | 奖励 |

**危险信号**
- 超过7天未登录
- 反馈中出现"算了""太麻烦"等词
