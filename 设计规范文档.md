# VideoLingo 设计规范文档

## 1. 项目概述

VideoLingo 是一个跨境电商 AI 视频本地化平台，帮助商家将视频内容快速翻译、配音并适配到目标市场语言，提升跨境营销效率。

## 2. 页面结构

| 路由 | 页面 | 功能说明 |
|------|------|----------|
| `/` | Landing | 产品介绍落地页，展示核心功能与定价 |
| `/upload` | 上传 | 视频文件上传，支持拖拽和格式校验 |
| `/generate` | 生成 | AI 翻译与配音生成，显示处理进度 |
| `/preview` | 预览 | 生成结果预览，支持字幕编辑和时间轴调整 |
| `/dashboard` | 仪表盘 | 项目管理，查看历史记录和使用统计 |
| `/login` | 登录 | 用户登录 |
| `/register` | 注册 | 用户注册 |

## 3. 设计 Token

### 颜色

| 用途 | 色值 |
|------|------|
| 主色 | `#6366f1` (indigo) |
| 主色 hover | `#4f46e5` |
| 成功 | `#22c55e` |
| 警告 | `#f59e0b` |
| 错误 | `#ef4444` |
| 背景（亮） | `#ffffff` |
| 背景（暗） | `#0a0a0a` |
| 文字主要 | `#171717` |
| 文字次要 | `#737373` |
| 边框 | `#e5e5e5` |

### 字体

```
font-family: system-ui, -apple-system, sans-serif;
```

### 圆角

- `lg`: 8px
- `xl`: 12px

### 间距

基准单位：4px（所有间距为 4 的倍数）

## 4. 组件规范

### Button

```css
/* 主按钮 */
background: #6366f1;
color: #ffffff;
padding: 8px 16px;
border-radius: 8px;
font-weight: 500;
hover: background #4f46e5;

/* 次按钮 */
background: transparent;
border: 1px solid #e5e5e5;
color: #171717;

/* 禁用态 */
opacity: 0.5;
cursor: not-allowed;
```

### Card

```css
background: #ffffff;
border: 1px solid #e5e5e5;
border-radius: 12px;
padding: 24px;
```

### Input

```css
border: 1px solid #e5e5e5;
border-radius: 8px;
padding: 8px 12px;
font-size: 14px;
focus: border-color #6366f1;
placeholder-color: #737373;
```
